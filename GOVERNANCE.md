# Documentation Governance

The Faramesh documentation is governed as part of the Faramesh open source
project.

Project-wide governance:

https://github.com/faramesh/faramesh-core/blob/main/GOVERNANCE.md

Documentation changes should keep the open source runtime, policy language,
registry, SDKs, and deployment guidance accurate. Product or service-specific
content must be clearly optional and must not imply that the open source runtime
requires a proprietary control plane.
