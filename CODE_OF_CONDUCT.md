# Code of Conduct

Participation in Faramesh documentation follows the Faramesh project Code of
Conduct.

Report conduct concerns to support@faramesh.dev. Security reports should go to
security@faramesh.dev.

The canonical project Code of Conduct is maintained in the Faramesh core
repository: https://github.com/faramesh/faramesh-core/blob/main/CODE_OF_CONDUCT.md
