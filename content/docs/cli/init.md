---
title: faramesh init
description: Scaffold governance.fms from your project layout.
---

```bash
faramesh init [--dir DIR] [--offline] [--non-interactive] [--yaml | --json]
```

Creates `governance.fms` with framework detection, starter rules, registry imports, and trust roots. Does **not** start the daemon.

See [faramesh init](/init/) for detection rules and examples. Next: [`faramesh check`](/cli/check/) then [`faramesh dev`](/cli/dev/) or [`faramesh apply`](/cli/apply/).
