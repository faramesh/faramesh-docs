---
title: What goes in governance.fms?
---

Top-level blocks: `import`, `runtime`, `provider`, `identity`, `trust`, `agent`.

Agent extensions include `rate_limit`, `redact`, `egress`, `model_policy`, `session`, `spawn`, `completion_gate`, `enforcement`, and `alert`.

See [FPL reference](/fpl/) for syntax. YAML and JSON equivalents use the same fields.
