## Faramesh

Faramesh governs AI agent tool calls before they execute.

```bash
faramesh init
faramesh dev
faramesh apply
```

A **stack** is a directory with `governance.fms` (FPL, YAML, or JSON). `faramesh apply` compiles it and starts the daemon.

A **provider** is a `provider "name" { }` block. The daemon calls it over gRPC for secrets, identity, KMS, audit sink, and cost.

A **policy pack** is an import from `registry.faramesh.dev` with a pinned version.

### Key capabilities

- **Governance as code** — one file per stack; `faramesh check` / `plan` / `apply` replace legacy `serve` flags.
- **Deterministic engine** — no LLM judges on the enforcement path; structured denials to adapters.
- **Evidence** — WAL-first DPR chain; `faramesh audit verify` walks active and archived segments.
- **Registry** — providers, policy packs, and framework profiles with pinned semver imports.
