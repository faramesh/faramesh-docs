# Security Policy

Do not open public issues for vulnerabilities in Faramesh documentation,
examples, install snippets, or generated site assets.

Preferred reporting paths:

- GitHub Security Advisory: https://github.com/faramesh/faramesh-docs/security/advisories/new
- Email: security@faramesh.dev

Reports should include the affected page or asset path, impact, reproduction
steps, and any suggested remediation.

Security reports are acknowledged within 3 business days and receive an initial
assessment within 7 business days.
